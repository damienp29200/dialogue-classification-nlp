# Dialogue Classification using NLP

This project focuses on the automatic classification of dialogues using natural language processing (NLP) and machine learning techniques.

## Objective

The goal is to classify dialogues into predefined categories by applying text preprocessing and supervised learning models.

## Methods

The project includes the following steps:

- Text preprocessing
- Tokenization
- Feature extraction
- Training and evaluation of classification models

## Technologies

- Python
- pandas
- scikit-learn
- nltk

## Dataset

The dataset used for this project is not included in this repository.
